Instalacion

1- Correr npm install (instala dependencias)
2- Correr comando npm start (inicia aplicacion en el puerto 3000)
3- Crear un archivo de nombre .env en la raiz del repositorio, aqui se guardaran las variables de entorno
4- Crea un archivo .gitignore en la raiz del repositorio, agregar aqui archivos a ignorar:
package-lock.json
.env
node_modules/
.gitignore
